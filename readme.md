# FLAME Fitness Club Website

Welcome to the official repository for the **FLAME Fitness Club** website. This platform is designed to showcase our club's activities, team, events, and sponsorship opportunities, providing an immersive experience for our community.

## Built With

This project leverages modern web technologies to deliver a high-performance and visually stunning experience:

- **HTML5**: Semantic structure and content.
- **Tailwind CSS**: Utility-first CSS framework for rapid and responsive styling.
- **JavaScript (ES6+)**: Dynamic interactivity and logic.
- **Three.js**: Immersive 3D background elements and character animations.
- **Vite**: Next-generation frontend tooling.

## Features

- **Immersive 3D Experience**: Interactive 3D character models and environments powered by Three.js.
- **Responsive Design**: Fully optimized layouts for Desktop, Tablet, and Mobile devices.
- **Dynamic Navigation**: Smooth page transitions without full reloads.
- **Interactive UI**: Engaging animations, hover effects, and a modern dark-themed aesthetic.
- **Sections**:
  - **Home**: Welcome screen with 3D interactions.
  - **Our Team**: Meet the trainers and executive committee.
  - **Events**: Upcoming and past club events.
  - **Sponsorship**: Information for potential partners.
  - **About**: Learn more about the club.

*Empowering the FLAME community through fitness and wellness.*
